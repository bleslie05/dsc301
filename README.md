## üß† Topics Covered  

- **Database Design:** normalization, keys, schema creation  
- **SQL Querying:** `SELECT`, `JOIN`, `GROUP BY`, `HAVING`, `ORDER BY`  
- **Aggregate & Scalar Functions:** `AVG`, `SUM`, `COUNT`, `UPPER`, `SUBSTR`, `REPLACE`  
- **Joins & Relationships:** inner, left, and outer joins; primary/foreign key constraints  
- **Data Cleaning:** string operations and conditional filtering  
- **Numeric Queries:** aggregation, rounding, and revenue computation  
- **SQLite in R:** querying via `DBI` and `RSQLite`  
- **Quarto Integration:** combining SQL queries and text output into reproducible reports
